# Reverse Array In-Place

Implement a program that **reverses an array** of 32‑bit floating
point numbers **in-place**. The program should take an input array and
reverse the order of its elements directly in the same array.

---

## Implementation Requirements

- Use only native features (no external libraries).
- The `solve` function signature **must remain unchanged**.
- The final result must be stored back in the array **`input`**.

---

## Examples

### Example 1

**Input**

- `input = [1.0, 2.0, 3.0, 4.0]`

**Output**

- `input = [4.0, 3.0, 2.0, 1.0]`

---

### Example 2

**Input**

- `input = [1.5, 2.5, 3.5]`

**Output**

- `input = [3.5, 2.5, 1.5]`

---

## Constraints

- Let \(N\) be the length of the array.
- The constraints are:

	\[
	1 \leq N \leq 100{,}000{,}000
	\]

- All elements are 32‑bit floating point numbers (`float`).
